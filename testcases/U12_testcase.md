### **TC_ALERTS_001: Работа с алертами (Input Alert)**

**Цель:** Проверить корректную работу диалогового окна типа Input Alert: ввод пользовательского имени и применение этого имени на странице в виде приветственного текста.

**Предусловие:**
1. Открыт браузер.
2. Обеспечен доступ к интернету.
3. Осуществлен переход на страницу с примерами Alert (URL задаётся как `ALERTS_URL`).

**Тестовые данные:**
- Имя пользователя для ввода в алерт:`"AutoTestName"`.

**Шаги:**
1. Найти и нажать элемент **"Input Alert"** на странице.
2. Переключиться во фрейм с демо для Input Alert и убедиться, что кнопка вызова алерта доступна и отображается.
3. Нажать кнопку вызова **Input Alert** во фрейме.
4. Убедиться, что открылось диалоговое окно типа Input Alert с полем ввода.
5. Ввести в поле алерта тестовое имя.
6. Подтвердить ввод (нажать кнопку подтверждения в алерте).
7. Найти элемент на странице, в котором должен отобразиться результат (текст приветствия с введённым именем).
8. Считать фактический текст из этого элемента.

**Ожидаемые результаты:**
*   После нажатия на ссылку **"Input Alert"** и переключения во фрейм становится доступна кнопка вызова Input Alert.
*   Диалоговое окно Input Alert успешно открывается после нажатия кнопки вызова.
*   Введённое в алерт имя `name` успешно принимается после подтверждения.
*   На странице в ожидаемом элементе отображается текст приветствия строго в формате:  
    `Hello {name}! How are you today?`,  
    где `{name}` это то значение, которое было введено в алерт.

**Постусловие:**
*   Закрыть браузер.
