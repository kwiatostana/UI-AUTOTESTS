### **TC_COOKIES_AUTH_001: Авторизация с использованием Cookies**

**Цель:** Проверить механизм сохранения сессии (cookies) в файл и восстановления сессии без повторного ввода учетных данных.

**Предусловие:**
1. Открыт браузер.
2. Известны валидные учетные данные для сайта: `https://www.sql-ex.ru/` (COOKIES_LOGIN, COOKIES_PASSWORD).

**Тестовые данные:**
- Login: COOKIES_LOGIN
- Password: COOKIES_PASSWORD
- Cookie file: COOKIE_FILE_PATH

**Шаги:**
1. Открыть страницу авторизации `https://www.sql-ex.ru/`.
2. Ввести **Login**.
3. Ввести **Password**.
4. Нажать кнопку **Enter**.
5. Проверить успешную авторизацию по наличию ссылки **Logout**.
6. Сохранить текущие куки браузера в файл `state/cookies.json`.
7. Проверить, что файл `state/cookies.json` был создан.
8. Очистить все куки браузера.
9. Проверить, что пользователь **не авторизован** - ссылка Logout отсутствует.
10. Загрузить куки из файла `state/cookies.json` в браузер.
11. Обновить страницу.
13. Проверить, что пользователь **авторизован** - ссылка Logout присутствует.

**Ожидаемые результаты:**
*   После первичного входа по логину и паролю пользователь успешно авторизован.
*   Файл с куками `state/cookies.json` создан и содержит данные сессии.
*   После очистки куков пользователь не авторизован.
*   После загрузки куков из файла пользователь автоматически авторизован без повторного ввода учетных данных.

**Постусловие:**
*   Удалить файл `state/cookies.json`.
